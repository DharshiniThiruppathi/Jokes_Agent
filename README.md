# Jokes_Agent
An AI-powered Joke Agent that generates random jokes, explains them if needed, and can tell jokes based on different categories using an LLM (OpenRouter).

📌 Features

😂 Generate random jokes
🎭 Category-based jokes (Programming, Dad, Dark, Knock-Knock, etc.)
💡 Explain the joke
🔄 Generate another joke on request
🤖 Uses an LLM through OpenRouter
🛠️ Easy to customize and extend
